# Node.js Express MVC Starter
## Getting Started
### Installing
Windows cmd or linux terminal
```
git clone https://github.com/Phoique/express-mvc-starter .
```
### Package install
`
npm install
`
### Start project (Do not forget to run MongoDB)
```
npm run start:dev
```
### Type localhost in the web browser url section
```
http://localhost:3000
```
